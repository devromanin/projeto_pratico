Projeto Prático – Agenda de Contatos 

Uma aplicação simples em Python para gerenciar uma agenda de contatos. Com ela, você pode **adicionar, visualizar, atualizar, marcar como favorito e excluir contatos**. Ótimo para estudar manipulação de dados em listas/dicionários e ganhar prática com lógica procedural e I/O em Python.

Funcionalidades

Criar novo contato (nome, telefone, e-mail, etc.)  
Listar todos os contatos salvos  
Atualizar dados de um contato existente  
Excluir contatos  
Marcar/desmarcar contatos como “favoritos”  

Como usar / Executar localmente

Clone o repositório digitando no terminal:
git clone https://github.com/devromanin/projeto_pratico.git

Entre na pasta do projeto:
cd projeto_pratico

Execute o script principal:
python Agenda_contatos.py

A aplicação vai abrir no terminal e você só segue o menu que aparece.
